# AI Prompts Used – Prodesk IT Internship Project (Dev Detective)

The following AI prompts were used during the development of this project strictly for
learning, debugging, and conceptual understanding purposes.
All code was reviewed, understood, and manually written or integrated by the developer.

---

### Prompt 0 (Project Understanding & Planning)

Explain the Week 3 internship task requirements and create a clear roadmap to build a GitHub User Search and Comparison app using Fetch API and JavaScript.

---

### Prompt 1 (API Basics)

What is an API? Explain APIs in simple terms with a real-world analogy and how frontend applications use them.

---

### Prompt 2 (Fetch API & Async/Await)

Explain how the JavaScript Fetch API works with async and await, including how promises are resolved.

---

### Prompt 3 (GitHub User API)

How do I fetch GitHub user profile data using the GitHub REST API and display it in a JavaScript application?

---

### Prompt 4 (Error Handling – 404)

How can I handle a 404 error when a GitHub username does not exist and show a user-friendly error message without crashing the app?

---

### Prompt 5 (Loading State UX)

How can I show a loading spinner or text while data is being fetched from an API and hide it once the response is received?

---

### Prompt 6 (Fetching Repositories)

How do I fetch a user's repositories using the `repos_url` from the GitHub API and work with the returned JSON data?

---

### Prompt 7 (Top Repositories Logic)

How can I sort GitHub repositories by latest update and display only the top 5 repositories in JavaScript?

---

### Prompt 8 (Date Formatting)

The GitHub API returns dates in ISO format. How can I convert and format dates into a human-readable format like "25 Jan 2023"?

---

### Prompt 9 (Clickable Links)

How can I make repository names clickable so they open the GitHub repository in a new browser tab?

---

### Prompt 10 (Battle Mode Logic)

How can I fetch data for two GitHub users simultaneously using `Promise.all()` and compare them based on stars or followers?

---

### Prompt 11 (Total Stars Calculation)

How can I calculate the total number of stars across all repositories of a GitHub user?

---

### Prompt 12 (Conditional UI Styling)

How can I dynamically apply different CSS classes to highlight a winner and loser card based on comparison results?

---

### Prompt 13 (UX Enhancement – Motivational Messages)

How can I display random motivational messages for winners and encouraging messages for losers after each comparison?

---

### Prompt 14 (Clipboard & Sharing)

How can I copy dynamically generated content to the clipboard using JavaScript for sharing results?

---

### Prompt 15 (Git & GitHub Submission)

Guide me on Git and GitHub commands to initialize a repository, commit changes, and push my project for internship submission.
